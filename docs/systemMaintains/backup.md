## DISM++
1. 进入 WinPE 后，打开 DISM++ 工具；  
![](./img/DISM++.png)
2. 打开后会提示用户协议，点击“接受”按钮 
![](./img/DSIM++_2.png)
3. 在左下角菜单中选择“恢复功能”-“系统备份”  
![](./img/DSIM++_3.png)
4. 选择系统备份路径后点击“确定按钮”
![](./img/DSIM++_4.png)
5. 耐心等待系统备份完成
